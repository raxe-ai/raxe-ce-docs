## Quick Comparison

| Integration | Best For | Scan Points | Install |
|-------------|----------|-------------|---------|
| **MCP Gateway** | Claude Desktop, Cursor | Tool calls, responses | `pip install raxe[mcp]` |
| **LangChain** | Agent apps, RAG | Prompts, tools, memory | `pip install raxe[langchain]` |
| **LiteLLM** | Multi-provider apps | All LLM calls | `pip install raxe[litellm]` |

| **CrewAI** | Multi-agent crews | Agent messages, tasks | `pip install raxe[crewai]` |

## Need Help Choosing?

<AccordionGroup>
  <Accordion title="I'm using Claude Desktop or Cursor">
    Use the **MCP Gateway** - it acts as a transparent security proxy for any MCP server, scanning all tool calls and responses without modifying your existing setup.
  </Accordion>
  <Accordion title="I'm building a RAG application">
    Use **LangChain** or **LlamaIndex** depending on your framework. Both provide deep integration with document retrieval and query pipelines.
  </Accordion>
  <Accordion title="I'm using multiple LLM providers">
    Use **LiteLLM** - it provides a unified interface for 100+ providers with RAXE protection built in.
  </Accordion>
  <Accordion title="I just want to protect OpenAI calls">
    Use the **OpenAI Wrapper** - it's a drop-in replacement with zero code changes beyond the import.
  </Accordion>
  <Accordion title="I need enterprise SIEM integration">
    See our [SIEM documentation](/integrations/siem) for Splunk, CrowdStrike, Sentinel, and CEF integrations.
  </Accordion>
  <Accordion title="I'm building multi-agent systems">
    Use **CrewAI** or **AutoGen** depending on your framework. Both support inter-agent message scanning.
  </Accordion>
</AccordionGroup>
